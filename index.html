Movie Suggestion System - Python

  • Using SpaCy and OOP I created a movie suggestion system. The user inputs the description of a movie they like. Using NLP my program checks the similarity of this description against a set list of movies. The movie with the highest similarity to the user input is the movie suggestion provided to the user.

  • This project was built with the goal of providing a task management system to the user. Admin's can add users, add new tasks and assign tasks to users. All users can view all tasks while tasks can only be edited by admins. Admins can also produce both a task overview and user overview report which creates a txt file with statistics about tasks and individual users.
